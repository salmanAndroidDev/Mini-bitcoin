# Mini-bitcoin
Mini-bitcoin is a simple blockchain Dapp (Decentralized web application).<br>
to run nodes type below commands<br>
`npm run node_1`<br>
`npm run node_2`<br>
`npm run node_3`<br>
`npm run node_4`<br>
`npm run node_5`<br>
